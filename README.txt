Запуск

java -jar Parser-1.0.jar <Файл чтения> <Файл записи>

<Файл чтения> - абсолютный путь к бинарному файлу источнику данных
<Файл записи> - абсолютный путь к текстовому файлу сохранения результатов обработки

Привечание: Если <Файл записи> уже существует, его содержимое будет перезаписано. В противном случае файл будет создан.

Для демонстрации парсера, используйте файл с данными (file) в корневом каталоге проекта.
